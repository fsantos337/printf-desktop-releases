## What is PrintF?

PrintF is a free portable desktop tool built for QA and software testing teams. It centralizes the entire evidence documentation process — from screen capture and image annotation to automatic document generation — in a single `.exe` with no installation required.

## ✨ Features

| Module | Description |
|---|---|
| 📸 Evidence Capture | Manual or automatic screenshot capture with session management |
| ⚡ Quick Capture | Instant screenshot via keyboard shortcut, even from the system tray |
| 🖊️ Image Editor | Rectangle, circle, arrow, text, blur and numbered circle tools |
| 🎥 Video Recording | Record full screen, region or active window with countdown and floating bar |
| 📋 Template Generator | Batch document generation from CSV, XLSX, ODS or JSON test case files |
| 📄 Output Formats | DOCX (default) or ODT — configurable per project |
| 🖨️ PDF Converter | Convert DOCX/ODT to PDF using LibreOffice |
| 🗑️ File Cleanup | Remove unnecessary evidence files by type |
| 📊 Reports | Session statistics and document generation history |

## 🚀 Getting Started

1. Download the latest portable `.exe` from [Releases](../../releases)
2. Copy the folder to any location — no installation needed
3. Run `PrintF.exe`
4. On first run, `printf.db` is created automatically in the same folder — keep it between updates

## 📋 Requirements

- Windows 10 or higher
- Minimum resolution 1024×600
- [LibreOffice](https://www.libreoffice.org/) *(required for PDF conversion, ODT templates and H.264 video)*

Sample templates and test case files are included in the `docs/` folder.
